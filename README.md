## Installation

Support python version 3.5 later

### pip install


Version latest
```
$ pip install https://github.com/benchuang11046/afs.git
```

### From sources


To build the library run:
```
$ python setup.py install
```

To build the wheel package:
```
$ python setup.py bdist_wheel
```
